# cli-todo-app

## simple cli todo app in Go

### list all todos:
    $ ./todo -list

### add a todo to the list:
    $ ./todo -add [todo name]

### mark todo as completed:
    $ ./todo -complete=[index]

### delete a todo from the list:
    $ ./todo -del=[index]
